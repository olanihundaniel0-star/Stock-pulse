# StockPulse

A full-stack inventory management system for retailers and wholesalers.
Built to handle real business operations — stock tracking, role-based
access, and transaction history — with a clean, responsive interface.
 
**Backend:** https://github.com/olanihundaniel0-star/Stock-pulse-backend

## Tech Stack

React 19, TypeScript, Vite, Tailwind CSS, Recharts, Supabase

## Features

- Role-based access control (Admin and Staff roles)
- Product inventory with SKU tracking, categories, and stock levels
- Stock in/out operations with full audit trail
- Dashboard with real-time metrics and stock movement charts
- Transaction history with user attribution
- Dark mode support
- Offline awareness

## Getting Started
```bash
git clone https://github.com/olanihundaniel0-star/Stock-pulse
cd Stock-pulse
npm install
npm run dev
```
